# LMS Portal

A **Learning Management System (LMS)** focused on the **student side of portal**.  
Easily view courses, lessons, and submit answers.

---

##  Features

- View enrolled courses
- Track assignments
- View lessons and lesson materials
- Submit response via code, upload or simply mark as done.
